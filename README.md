# JCI Drones – Next-Gen Aerial Intelligence

A fictional drone company website built as an HCI Lab assignment at BINUS University.

## Pages
- **Home** – Hero banner, featured drones, innovation awards, fields of expertise
- **Products** – Product catalog with 6 drone models
- **Forum** – Community forum with category filters
- **Store** – Official app, e-commerce partners, physical store locator
- **Contact Us** – Contact form with JavaScript validation

## Tech Stack
HTML, CSS, JavaScript
